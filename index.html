Work

As part of a full-stack web development course completed in 2019, I was required to build an e-commerce website from scratch, utilising HTML, CSS and PHP, which then had to be migrated to Laravel once completed.
The site allows users to register, log in, browse for different products, send enquiries, add and remove products from shopping cart and complete purchases, with all data stored on a MySQL database. From an Admin point of view, a basic CMS panel was also built from scracth and allows the administrator to add, update and remove product information.
This site was hosted locally and developed for presentation and completion of the course only and therefore it is not a live website, however a demo video can be found here. Access to the code is also available thorugh my Github profile in the contact page.

LIFT is an online platform I am currently developing for a personal trainer, sports scientist and good friend of mine in Argentina. His company works with and trains elite athletes across different sports, and the aim of this platform is to store and track their performances.
From a user's perspective, athletes will be able to register, create their profile, and upload training information through forms using their personal phones or iPads available at the LIFT gym.
From the admin side, it will run a number of tests on different aspects of the athletes' training, and display their progress in the form of charts as part of the user's profile. This data can then be compared and benchmarked against the average for a certain team, sport, or position.
This site is also Laravel based and includes the utilisation of Charts.js and LaraCharts. It is still work in progress, however a demo of its current functionality can be found here and access to the code is also available through my Github profile.

Mmarketing is a simple, one-pager brochure site that I am currently building for my friend, a marketing specialist based in Bali. This was built using HTML, CSS and Bootstrap, and allows the customer to use the navigation bar or scroll down the page to see company information and services offered, as well as a contact form where they can leave an enquiry.
This site is awaiting further content and photography to be provided and is expected to be published within the next month. Access to the code is available on my Github profile.
